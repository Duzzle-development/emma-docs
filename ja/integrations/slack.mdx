# Slack インテグレーション

ワークフローを離れることなく、Slack で直接 Emma とチャットしてヘルプを得ましょう。

## はじめに

### インストール

<Steps>
  <Step title="設定に移動">
    Emma ダッシュボードで **設定 > インテグレーション > Slack** に移動します。
  </Step>
  <Step title="接続をクリック">
    「Slack に接続」ボタンをクリックします。
  </Step>
  <Step title="認証">
    Emma が Slack ワークスペースにアクセスすることを許可します。
  </Step>
  <Step title="チャット開始">
    任意のチャンネルで @Emma をメンションして会話を開始します。
  </Step>
</Steps>

## 使い方

インストール後、いくつかの方法で Emma と対話できます：

### Emma をメンション

任意のチャンネルやスレッドで **@Emma** をタグ付けして、質問したりヘルプを求めたりします。

```
@Emma Node.js で認証を実装するにはどうすればいいですか？
```

### ダイレクトメッセージ

プライベートな会話のために Emma にダイレクトメッセージを送信します。

### スラッシュコマンド

| コマンド | 説明 |
|---------|------|
| `/emma ask [質問]` | Emma に簡単な質問をする |
| `/emma help` | 利用可能なコマンドを表示 |
| `/emma status` | Emma のステータスを確認 |

## 権限

Emma には以下の Slack 権限が必要です：

| 権限 | 目的 |
|------|------|
| `chat:write` | Emma としてメッセージを送信 |
| `channels:history` | Emma がメンションされたチャンネルのメッセージを読む |
| `im:history` | ダイレクトメッセージを読む |
| `users:read` | パーソナライズされた応答のためにユーザーを識別 |

<Note>
Emma は直接メンションされたメッセージまたはダイレクトメッセージの会話のみにアクセスします。
</Note>
