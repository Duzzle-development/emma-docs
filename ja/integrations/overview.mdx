# インテグレーション

emma はお気に入りのツールと統合して、ワークフローにシームレスにフィットします。

## 利用可能なインテグレーション

<CardGroup cols={2}>
  <Card title="Slack" icon="slack" href="/ja/integrations/slack">
    Slack ワークスペースで直接 emma とチャット。
  </Card>
  <Card title="GitHub" icon="github">
    コードアシスタンスのためにリポジトリに emma を接続。
  </Card>
  <Card title="VS Code" icon="code">
    IDE で直接 emma を使用。
  </Card>
  <Card title="API" icon="webhook" href="/ja/api-reference/overview">
    REST API でカスタムインテグレーションを構築。
  </Card>
</CardGroup>

## インテグレーションのセットアップ

<Steps>
  <Step title="インテグレーションを選択">
    上記のリストから接続したいツールを選択します。
  </Step>
  <Step title="認証">
    選択したインテグレーションの認証手順に従います。
  </Step>
  <Step title="設定">
    ワークフローに合わせてインテグレーション設定をカスタマイズします。
  </Step>
  <Step title="使用開始">
    接続したツール内で emma の使用を開始します。
  </Step>
</Steps>

## 別のインテグレーションが必要ですか？

<Note>
必要なインテグレーションが見つかりませんか？お知らせください！ユーザーのフィードバックに基づいて常に新しいインテグレーションを追加しています。
</Note>

インテグレーションのリクエストは [support@emma.ai](mailto:support@emma.ai) までご連絡ください。
