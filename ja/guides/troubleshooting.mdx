# トラブルシューティング

問題がありますか？以下でよくある問題の解決策を見つけてください。

## よくある問題

<AccordionGroup>
  <Accordion title="Emma が応答しない">
    **考えられる原因：**
    - ネットワーク接続の問題
    - サービスが一時的に利用不可
    
    **解決策：**
    1. インターネット接続を確認
    2. Emma サービスのステータスを確認
    3. ページを更新するかアプリケーションを再起動
  </Accordion>
  
  <Accordion title="予期しない出力">
    **考えられる原因：**
    - 曖昧な指示
    - コンテキスト不足
    
    **解決策：**
    1. より具体的な指示を提供
    2. 関連するコンテキストと例を含める
    3. 複雑なリクエストを小さなステップに分割
  </Accordion>
  
  <Accordion title="認証エラー">
    **考えられる原因：**
    - 資格情報の期限切れ
    - 無効な API キー
    
    **解決策：**
    1. API キーが正しいか確認
    2. 資格情報が期限切れでないか確認
    3. 必要に応じて API キーを再生成
  </Accordion>
</AccordionGroup>

## バグの報告

問題を報告する際は、以下を含めてください：

- 問題の明確な説明
- 問題を再現する手順
- 期待される動作と実際の動作
- 受け取ったエラーメッセージ
- 環境の詳細（ブラウザ、OS など）
