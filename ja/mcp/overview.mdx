## MCPとは？

**Model Context Protocol（MCP）** は、AIアプリケーションが外部のデータソースやツールに安全に接続するためのオープンスタンダードです。MCPは、AIアプリケーションにとってのUSB-Cポートのようなもの — 様々なサービスに接続するための標準化された方法です。

MCPを使用することで、emmaのようなAIエージェントに、統一されたインターフェースを通じてツール、データベース、API、その他のサービスへのアクセスを提供できます。

## 対応トランスポート

emma MCP Serverは現在、**stdio（標準入出力）** トランスポートに対応しています。これはCursorなどのローカルMCPクライアントで使用されます。

| トランスポート | 状態 | 説明 |
|--------------|------|------|
| **stdio** | 対応 | ローカルプロセスとして起動し、stdin/stdoutで通信 |
| SSE | 未対応 | Server-Sent Events |
| Streamable HTTP | 未対応 | HTTPベースのストリーミング |

## 対応クライアント

現在、emma MCP Serverは以下のクライアントを公式にサポートしています：

- **Cursor** — AI搭載コードエディタ
